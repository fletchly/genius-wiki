# Installation

Download and install Genius

## Prerequisites

- A Minecraft server running Paper 1.21+

## Installation steps

### 1. Download the plugin

Download the latest version of Genius from:

- [Modrinth](https://modrinth.com/plugin/genius)
- [Github Releases](https://github.com/fletchly/genius/releases)

### 2. Stop your server

Stop your Minecraft server completely before installing the plugin to prevent any issues.

### 3. Upload the plugin

1. Navigate to your server's `plugins` folder
2. Move the downloaded `Genius-X.X.X.jar` file to the `plugins` folder

### 4. Start Your Server

Start your Minecraft server. The plugin will automatically generate its configuration files on first launch.

### 5. Verify installation

Check your server console for a message confirming the plugin has loaded successfully. You should see something like:

```
[Genius] Successfully enabled Genius vX.X.X!
```
