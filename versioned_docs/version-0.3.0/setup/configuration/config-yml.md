# config.yml

As of genius v0.3.0, YAML configuration in no longer supported. Please see the [genius.conf](genius-conf.md) page for reference on the new configuration.

To migrate to the new config format, install Genius 0.3.0+ and restart your server. then, copy values from your old config.yml to the newly created genius.conf file.
