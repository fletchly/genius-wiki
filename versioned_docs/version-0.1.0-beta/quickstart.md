# Quickstart

Get genius running in 2 steps.

## Steps

### 1. Install genius

Follow the [installation guide](setup/installation.md#installation-steps) to download Genius and install it on your server

### 2. Set up cloud hosting

Follow the [cloud hosting guide](setup/hosting.md#cloud-hosting-ollamacom) to set up (free!) cloud hosting for your LLM on ollama.com.
